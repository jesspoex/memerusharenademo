# Memerush — Solana PvP Trading Battle Platform

Memerush is a gamified trading platform that introduces a competitive PvP layer on top of trading.

Users can compete in real-time trading battles, climb leaderboards, and turn trading performance into a social and competitive experience.

## 🚀 Overview

Most trading platforms today are passive and lack engagement.

Memerush solves this by transforming trading into a competitive game:
- Users join PvP trading battles
- Compete based on performance
- Climb a public leaderboard
- Build reputation and rank

This creates a new interaction layer combining trading, gaming, and social competition.
